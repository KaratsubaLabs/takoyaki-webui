# タコヤキ Takoyaki Web UI
A VueJS frontend for からつばLABS' **takoyaki** VPS platform

## RUNNING FOR DEVELOPMENT

### Compiles and hot-reloads for development

```
$ npm install
$ npm run dev
```

## RUNNNING FOR PRODUCTION

Build the frontend
```
$ npm run build
```
